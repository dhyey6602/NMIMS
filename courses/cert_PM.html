Program Features

Highlights:
  • This 6-month program teaches you the fundamental concepts of project management.
  • The program focuses on empowering you to effectively manage different types of projects, such as public, business, information systems, energy, transportation, engineering, and construction.
  • The courseware teaches you specific techniques that are required to carry out projects in a systematic and timely manner.
  • The program familiarises you with the key components of selecting, initiating, planning, executing, controlling, evaluating and terminating projects.
Outcomes:
  • Display skills and abilities for effective project management
  • Apply project management principles to existing projects
  • Understand the tools and techniques used in project management
  • Streamline and automate the project management process
